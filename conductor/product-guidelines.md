# Diretrizes do Produto - FOOCUS

## Tom de Voz: O Mentor Encorajador Estóico
O FOOCUS fala como um mentor que valoriza a disciplina acima de tudo, mas entende a jornada humana.
- **Direto e Estoico:** Comunica fatos e métricas sem rodeios. "Você concluiu 2/3" em vez de "Parabéns, você está indo bem!".
- **Encorajador:** Quando o usuário falha ou hesita, o tom muda para suporte estratégico. "A falha é um dado. Ajuste o plano e retome agora."
- **Foco no Agora:** Toda comunicação deve impulsionar o usuário para a próxima ação imediata.

## Diretrizes de Design (UI/UX)

### Bibliotecas de Componentes
- **Shadcn UI:** Base para componentes estruturais e utilitários.
- **HeroUI:** Componentes de interação rica e animações fluidas.

### Princípios Visuais
- **Design Moderno e "Soft":** Uso de bordas generosas (`rounded-[2rem]`) e sombras suaves.
- **Tipografia de Alta Legibilidade:** Foco em fontes claras para leitura diária, com pesos variados para hierarquia de informação.
- **Foco Radical:** Menos é mais. Espaçamento amplo para isolar prioridades.
- **Feedback Energético (Gamificação):** Emerald (Sucesso), Orange (Fogo/Streaks), Blue (Estratégia).

### Otimização para Mobile e Tablet (Desk Mode)
O FOOCUS deve ser pensado para uso híbrido, especialmente em dispositivos Android e tablets.
- **Layout Responsivo:** Adaptação fluida para telas menores, mantendo a facilidade de toque (touch targets amplos).
- **Pomodoro "Desk Companion":** A interface de foco/pomodoro deve ser otimizada para permanecer aberta em tablets ou celulares sobre a mesa.
    - Tipografia gigante para o cronômetro, visível à distância.
    - Controles simplificados e cores imersivas que mudam conforme o estado (foco vs. pausa).
    - Prevenção de bloqueio de tela opcional enquanto o timer estiver ativo.
