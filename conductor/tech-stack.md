# Stack Tecnológica - FOOCUS

## Frontend

- **Framework:** React 19 (Componentes funcionais e Hooks)
- **Linguagem:** TypeScript (Tipagem estrita para maior segurança)
- **Build Tool:** Vite (Performance de desenvolvimento ultrarrápida)
- **Roteamento:** React Router (Utilizando `HashRouter` para compatibilidade de
  deploy)

## Backend & Persistência (BaaS)

- **Provedor:** Supabase (PostgreSQL + Auth + Storage)
- **Projeto URL:** `https://qezagogqrdjzvlggaaxv.supabase.co`
- **Chave Pública:** `sb_publishable_<redacted>`
- **ORM/Client:** Supabase JS Client

## Inteligência Artificial

- **SDK:** Google Generative AI (@google/genai)
- **Modelo:** `gemini-3-flash-preview` (Utilizado para decomposição de metas e
  insights)

## Estilização e UI

- **Base:** Tailwind CSS (Utilitários de CSS modernos)
- **Componentes:**
  - **Shadcn UI:** Estrutura e acessibilidade.
  - **HeroUI:** Animações e componentes premium.
- **Ícones:** Google Material Symbols (Rounded)

## Infraestrutura e Ferramentas

- **Controle de Versão:** Git
- **Ambiente:** Node.js (Ambiente de execução local)
- **Variáveis de Ambiente:** Gerenciadas via `.env.local`
